# scholarship-informationsystem-webapplication

This is Scholarship Information System Web Application project for Web Enterprise Programming subject in 4th semerter

Features:
1. CRUD system
2. Login and Register System (Multirole: Administrator and Viewer)
3. Form Validation
4. Password Strength Meter
5. Data Visualization
6. User Activity Log


Privileges
1. Administrator (CRUD)
2. Viewer (R)
